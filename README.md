# UpdateOrCreate #
## DB utility plugin for ActiveRecord ##
update_or_create allows update_or_create to ActiveRecord.

### Install ###
* Add  

```ruby
gem 'update_or_create_jb'
```
  to Gemfile.

### Usage ###
* Model.update_or_create([id:1, value:'test']) or etc...

### System requirement ###
* Ruby 2.0 or later.
* RoR 4.1 or later.
